# Moonshot Competitive Intelligence Dashboard — Backend

This will be a FastAPI backend serving the React dashboard and exposing endpoints for scraped and analyzed data.

## Planned Structure
- main.py: FastAPI entrypoint
- api/: API route modules
- models/: Pydantic models
- services/: Business logic (scraper, analysis, etc.)

See Bestie_ai/BestieAdvice/backend and Bestie_ai/backend for FastAPI+LangGraph integration patterns.
